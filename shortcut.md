- vs

```
command +

b => EXPLORER出し入れ

f => ファイル内検索
f +shift =>　グローバル検索

d => 行検索


shift + o =>　コンポーネントに移動

+ :拡大
-　：縮小

f12 => 使われている先にとぶ
control +
- => go back　戻る(f12実行後)
- +shift => go forward　進む(f12実行後)
```

- iterm

```
control r => fzf

command d => 分割
command d shift => 縦分割

```

- chrome

```
command + f => 検索
shift + enter => 前に戻る
enter => 進む
command + click =>新しいタブで開く


```
